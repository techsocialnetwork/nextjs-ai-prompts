# Hello World

This is my first blog post, built entirely with **Next.js 14** and **AI assistance**.

## Why Next.js 14?

Next.js 14 introduces several game-changing features:

- **Server Components by default** — Better performance, smaller bundles
- **Server Actions** — Mutations without API routes
- **Partial Prerendering** — Best of static and dynamic
- **Turbopack** — Faster development builds

## The AI Workflow

This entire blog was created using AI prompts:

1. **Layout prompts** — Dashboard structure, navigation
2. **Data fetching prompts** — MDX parsing, static generation
3. **Styling prompts** — Tailwind configuration, dark mode
4. **Component prompts** — Cards, buttons, forms

## Code Example

Here's a simple Server Component:

```typescript
export default async function BlogPage() {
  const posts = await getAllPosts();
  
  return (
    <div>
      {posts.map(post => (
        <PostCard key={post.slug} post={post} />
      ))}
    </div>
  );
}
```

Notice how we can use `async` directly in the component — no `useEffect`, no client-side data fetching.
